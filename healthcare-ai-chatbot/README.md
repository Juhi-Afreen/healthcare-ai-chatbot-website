Plus HealthMate AI - Landing Page

A modern, responsive landing page for Plus Web Team's AI Chatbot service for healthcare providers.

Project Structure

healthcare-ai-chatbot/
│
├── index.html # Main entry point
│
├── assets/
│ └── images/ # Static images (logos)
│
├── css/
│ ├── main.css # Typography, vars, and base styles
│ └── animations.css # Keyframes and animation classes
│
├── js/
│ ├── main.js # Icon initialization
│ ├── navbar.js # Scroll effects and mobile menu
│ └── form-handler.js # Lead capture logic
│
└── components/ # HTML snippets for easy reference/inclusion
├── header.html
├── hero.html
├── features.html
└── footer.html

Features

Responsive Design: Mobile-first approach using Tailwind CSS.

Interactivity: Smooth scrolling, sticky navbar, and hover effects.

Custom Animations: CSS keyframes for floating elements and chat bubbles.

Lead Generation: Integrated form handler structure.

Setup

Open index.html in any modern web browser.

No build process required (standard HTML/CSS/JS).
